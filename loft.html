Website Commission: The Loft

A custom website built for The Loft, an artisanal coffee shop and community gathering space in Manitou Springs, CO. This project integrated accessible web development practices from the ground up — including semantic HTML, keyboard navigation, color contrast compliance, and screen reader-friendly structure — ensuring the site is usable and welcoming for all visitors.


Currently in active development pending final menu content, professional photography, and refined branding.

Loft Website (Draft)

Current Progress:

  • General layout for website structure
  • Placeholder text, images, and concepts
  • Established brand colors and logo
  • Interactive placeholder elements for menus, map, and feature pages
  • Responsive layout tested on desktop breakpoints

Design Decisions:

  • Branding developed using dominant colors inspired by the café’s signature blue espresso machine and the eclectic décor throughout the space
  • Navigation designed with colors drawn from The Loft’s iconic tie-dye merchandise
  • Typography selection using “Poppins” to reflect the café’s current menu design and maintain brand consistency
  • Layout optimized for readability and user flow, with clear visual hierarchy
  • Accessibility considerations applied to colors, font sizes, and spacing

Future Considerations:

  • Friendly and approachable brand voice to reflect the café’s welcoming, community-focused environment
  • Placeholder images replaced with professional photos (coming soon)
  • Custom fonts, logos, and social media badges specially built for the website
  • Update to brand colors and typography based on menu redesign
  • Responsive layout tested on mobile breakpoints
  • Integrate micro-interactions to highlight featured items and promotions
  • Update website copy to match the café’s storytelling and community engagement voice
